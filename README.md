## 🔧 Installation  

Run the following commands in **Linux/macOS**:  

```bash
git clone https://github.com/KevinKhemra007/Sms_Tool.git
cd Sms_Tool
chmod +x Sms.py
python3 Sms.py
```
Run the following commands in Window
```bash
git clone https://github.com/KevinKhemra007/Sms_Tool.git
cd Sms_Tool
python Sms.py
